# dlab_pikachu_volleyball
NCTU Dlab 2020 final project

### 成品

* 可操控皮卡丘與CPU打排球, 先得到5分者獲勝
* 選單畫面可以更換皮卡丘造型
* 可以用開關切換顯示按鍵提示
* CPU會跳殺(怕)
* 殺球時球會變色

------------------------------------
## 以下是分工流程

### 操控

* **力學**
  * 重力
  * 擊球造成的力
* **皮卡丘移動**
  * 左右、跳躍
  * 限定移動範圍
  * 殺球
* **球**
  * 限制飛行範圍
  * 球飛行軌跡
  * 判斷落地在哪一方
* **CPU**
  * 判斷如何行動
  
### 遊戲流程

* **場景間切換**
  * 開始畫面
  * 造型選擇畫面
  * 遊戲畫面
* **遊戲進行流程**
  * 開局
  * 得分
  * 比賽結束
* **算分**

### 顯示

* **圖片顯示**
  * 正確顯示
  * GIF速度控制
  * 大小控制
  * 重疊處理  
* **特效**
  * 彩蛋所需特效
* **美術**
  * 遊戲畫面美觀
  * 以少量reg裝飾畫面
